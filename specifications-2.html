Requirements

 On this section we explore the technical requirements for our game and Web UI

Main Requirements

Multiplayer chess tournament

Each tournament starts with 2 players, that play in pairs decided by the system. The tournament has a single-elimination format.

Play with other people

Inside a tournament a player is paired with another player to have a realtime game of chess

Monitor each tournament and its stats

On the Web UI the user is able to see the players on each tournament, the games that have taken place and statistics on the games played



All technical requirements

Chess Tourney Server

Chess Tourney needs at least one server with public access to internet so people can play

To play

People need to have executable file for specific OS on a computer with internet connection to be able to play the game.

Monitorization of a game

On a computer with internet connection, people can see a game, play by play by using the WEB UI

To use VR

People need a computer with VR capabilities and HTC Vibe
