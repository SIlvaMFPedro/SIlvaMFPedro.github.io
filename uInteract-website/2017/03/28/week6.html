WEEK 6

Posted 28 Mar 2017

Continuous development in each demonstration

Deviation of work effort to Avatar






As we approach the first of our demos deadline - Avatar - we found it necessary to move some of the team into the development of this demo in particular, since it needs to be fully functional at the end of this week. Therefore most of our workload and focus will be spent on the "Avatar" and "Post Under Investigation" demos which are the ones in more need of attention at this stage of development. We will share with you more information on this on the sections below.

Vertigo

Development suspended until Avatar demo is mostly functional

Avatar

Time was devoted into studying how to implement basic mechanics related to highlighting certain body parts of the custom Avatar. This comes directly connected to the joints being currently being worked upon on Blender and further implementation of a proper Mesh collider on Unity that will be adapted into the complete model, so that detailed highlights of movements made by the user can be possible to develop, in constrast to lighting some zones of the body that may be unrelated to the movement being currently made.

Dust Under Investigation

As the deadline approaches, this week's focus was to try a new solution related to the placement of UI aspects of the interface, after discussion with the client. We came to the conclusion it might not be the best solution yet, but decided to go along with it for the moment so that we can divert time into implementing the actual visual of the interface instead of relying on the prototypes we've been using the past few weeks.

Meanwhile, our version of the code related to the gestures of the Leap Motion sensor received further optimizations and it has come to a spot that we feel confortable enough to leave it suspended for the moment while we connect it with the interface so that we can begin testing with the actual UI the demo will be based upon instead of unity's basic modules. Pong

Not much to discuss in this week's report. Further study was made on how to extract information from Kinect's camera and how to calibrate that data using PCL in order to get accurate representations of the table in the spatial game's environment.

We will keep working and if everything goes as expected, should have a video of an actual representation of the Avatar demo next week so stay tuned for it. Thanks for reading!
