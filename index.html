What is Semi-Automatic Labelling?


This thesis is focused on the research of methods to register data into labels using a camera and LIDAR sensors in ATLASCAR 2, so that the vehicle can later create a model of the objects it will detect and follow. The detection, tracking and labelling system will combine the image data and the laser data from the sensors to create a semi-automatic tracking system that allows the user to associate an object category to the target and create datasets with metadata from the labelling.
